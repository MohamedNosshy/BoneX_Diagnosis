Welcome to our application

We are "without Bugs" team, and in this page, we will tell you about our project.

What is our app?

  • The app is a web application that utilizes pre-trained deep learning models for image classification.
  • Its goal is to provide users with a simple and intuitive interface to upload an image and obtain predictions about the content of the image using different pre-trained models.

What is the goal of our app?

  • The main goal of the app is to enable users to classify images into different categories or classes.
  • The app supports multiple models, each trained for a specific classification task.
  • Users can select the desired model type from the available options.
  • Once an image is uploaded and the model type is selected, the app uses the chosen pre-trained model to analyze the image and predict its class.
  • The predicted class label, along with the corresponding probability, is displayed to the user.

What does the application contain?

The application contains 9 models: (spine, chest, pelvis, Elbow, Finger, Forearm, Hand, Shoulder, and Wrist).

  • Spine model:

    • The spine model detect 3 possibilities (dislocation, fracture or normal).

  • Chest model:

    • The chest model detect 4 possibilities (flail, hemothorax, pneumothorax or normal).

  • Pelvis model:

    • The pelvis model detect 2 possibilities (fracture or normal).

  • Elbow model:

    • The elbow model detect 2 possibilities (fracture or normal).

  • Finger model:

    • The finger model detect 2 possibilities (fracture or normal).

  • Forearm model:

    • The forearm model detect 2 possibilities (fracture or normal).

  • Hand model:

    • The hand model detect 2 possibilities (fracture or normal).

  • Shoulder model:

    • The shoulder model detect 2 possibilities (fracture or normal).

  • Wrist model:

    • The wrist model detect 2 possibilities (fracture or normal).

  • Additionally, the code generates a Grad-CAM heatmap to visualize the regions of the image that contribute most to the prediction. The heatmap is saved and displayed on the web page along with the prediction results.
